1. Package Description

Package name: GlucoCheck

Class name: glucoCheckOps

How to import the package?

  1. Download the zip file of the package here.
  2. Extract the zip file into your desired directory
  3. Create your own Python file. This could also be a Jupyter notebook. (Make sure it is in the same directory)
  4. Write the following line of code to import the package: from GlucoCheck.glucoCheck import glucoCheckOps


How to Create an object?
  1. Import the package
  2. Write the following line of code: glucoCheckObject = glucoCheckOps()
