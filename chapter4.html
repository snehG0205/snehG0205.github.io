4. Helper Methods

These methods are used for internal processing. They are called by other functions in the class, especislly the variability indices that require data to be formatted in specific ways, for example, sampled at specific intervals or have the glucose values in a specific units.


4(a) Smoothing

This method performs a rolling-mean smoothening for a time-series to improve MAGE calculation. Rolling means (or moving averages) are generally used to smooth out short-term fluctuations in time series data and highlight long-term trends. This prevents tiny irrelevant excursions from adding noise to the MAGE score

Method Signature: def smoothing(self, data)

Function Parameters:

Name Description Datatype
data the dataset (CSV file) entered by the user to train the model. It should have the following format:
Display Time -> object
GlucoseValue -> float64
subjectId -> object
 pandas Dataframe


Return:A pandas DataFrame with smoothened glucose values

Example: Smoothening values




4(b) Subsampling

This function subsamples a time-series at 15 minute intervals. Our consolidated data is sampled at separate intervals such as 3-minutes, 5-minutes, 10-minutes and 15-minutes. Subsampling data at regular intervals ensured data is uniform.

GlucoseValue -> float64

Return:A pandas DataFrame with reading at 15 minute intervals
