5. Inner Methods

These methods are called only by the core method, not by the user.
They help in the reapetetive tasks required by various functions.


5.1 Plot Imputed Values

This method plots the graph for the imputed values for the model. It is called by the impute() method itself.

Method Signature: def plot(self, data)

Function Parameters:

Name Description Datatype
data the dataset (CSV file) entered by the user to train the model. It should have the following format:
Display Time -> object
GlucoseValue -> float64
subjectId -> object
 pandas Dataframe


Return:A plot of the imputed time series


5.2 Detect Gap

The detectGap method detects the GAP in a time series. It is called by the impute method itself.

Method Signature: def detectGap(self, testing_data):

Function Parameters:

Name Description Datatype
data the dataset (CSV file) entered by the user to train the model. It should have the following format:
Display Time -> object
GlucoseValue -> float64
subjectId -> object
 pandas Dataframe


Return:The starting and ending indexes of the gaps


5.3 Preprocess Date

This method is used to preprocess the data entered by the user. It identifies the date and converts it to the standard datetime format (%Y-%m-%d %H:%M:%S). It also converts the Timestamp to the index of the data frame.

Method Signature: def datePreprocess(self, data)

Function Parameters:

Name Description Datatype
data the dataset (CSV file) entered by the user to train the model. It should have the following format:
Display Time -> object
GlucoseValue -> float64
subjectId -> object
 pandas Dataframe


Return:The output is a pandas data frame of the csv file supplied with the 'Display Time' converted to the (%Y-%m-%d %H:%M:%S) format.
