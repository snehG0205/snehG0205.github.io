2. Constants

Constants are defined to store data. We have two defined constants, one to store our data and the other to store the metadata.
These constants are used to feed data to our learning model and to calculate teh glucose variability indices.



Constants
Name Description Datatype
consolidatedData Stores data accumulated from the CGMAnalysis package, Gluvarpro package, and the CGMAnalyzer packagedataset. pandas Dataframe
consolidated_meta Stores metadata for the data in 'consolidatedData' pandas Dataframe
